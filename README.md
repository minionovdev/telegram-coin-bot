# Telegram Coin Bot

Бот с системой коинов, топ игроков, ежедневными наградами и мини-играми.

## Команды

/bot - список команд
/balance - показать баланс
/top - топ игроков
/daily - ежедневная награда
/shop - магазин
/buy admin - купить админку
/bet 100 - ставка
/duel - дуэль (ответом на сообщение)

## Настройка

1. Добавить секрет BOT_TOKEN в Render
2. Развернуть репозиторий
3. Стартовая команда: python bot.py
